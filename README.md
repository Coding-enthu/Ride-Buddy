# Ride Buddy

## Project Description
Ride Buddy is a revolutionary ride-sharing application designed to connect passengers with drivers seamlessly. The platform focuses on providing a user-friendly experience and aims to enhance urban mobility by making ridesharing accessible and convenient.

## Features
- Quick ride request system
- Real-time tracking of rides
- Driver and passenger ratings
- Integrated payment systems
- User profiles and history

## Tech Stack
- **Frontend:** TypeScript, CSS, JavaScript, HTML
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Getting Started
### Prerequisites
- Node.js (latest version recommended)
- npm (Node package manager) or yarn

### Install and Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Coding-enthu/Ride-Buddy.git
   cd Ride-Buddy
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Configuration / Environment Variables
Create a `.env` file in the root directory and add the following variables:
- `DATABASE_URI`: Your MongoDB connection string
- `PORT`: Port number for the server

## Scripts
- `start`: Starts the development server
- `build`: Compiles the application for production
- `test`: Runs the tests
